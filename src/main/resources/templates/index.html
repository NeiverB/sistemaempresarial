Sistema de Gestión Farmacéutica Fybeca

Proyecto Integrador - Programación Orientada a Objetos con Java

Sector Empresarial

Farmacéutico y Comercial

Objetivo del Sistema

Desarrollar una plataforma que permita administrar medicamentos, clientes, pedidos, inventario y sucursales de Fybeca, optimizando los procesos internos y mejorando la atención al cliente.

Problema que busca resolver

La gestión manual de inventarios y pedidos puede generar errores, retrasos y falta de control sobre los medicamentos disponibles. El sistema permitirá centralizar la información y automatizar procesos.

Entidades Principales

  • Cliente
  • Medicamento
  • Pedido
  • Sucursal
  • Empleado
